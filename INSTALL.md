# Installatie instructies

## Stap 1: Installeer react-router-dom

```bash
npm install
```

Of specifiek:

```bash
npm install react-router-dom
```

## Stap 2: Start de development server

```bash
npm run dev
```

## Stap 3: Build voor productie

```bash
npm run build
```

---

**Note:** Alle code is klaar en werkt. Je hoeft alleen nog `npm install` te draaien om react-router-dom te installeren.
